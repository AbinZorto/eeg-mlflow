\begin{abstract}
This study evaluated a sparse low-montage resting-state EEG framework for identifying clinically interpretable biomarkers of response to transcranial direct current stimulation (tDCS) in adults with treatment-resistant depression from a fully remote home-based trial, under strict patient-level validation. Using 4-channel eyes-closed EEG, we ran a controlled sweep of 150 successful runs across two model families, an advanced hybrid 1D CNN-LSTM and a linear support vector machine (SVM), varying window size (2--10 s) and per-fold feature budget (\texttt{inner-k} 1--70) while applying leave-one-participant-out group equalization and SMOTE during training.

The strongest configuration was the hybrid model with 6 s windows and \texttt{inner-k=1}. It reached patient-level ROC-AUC 0.816, PR-AUC 0.579, balanced accuracy 0.786, accuracy 0.810, and MCC 0.571 over 21 held-out participants. The best SVM configuration (8 s, \texttt{inner-k=30}) reached ROC-AUC 0.510 and MCC 0.277. The hybrid winner selected 1 feature per fold and only 7 unique recurrent features overall, with a permutation-tested ROC-AUC advantage over chance ($p=0.008$). Hard-set overlap across folds was modest (mean pairwise Jaccard 0.190; Kuncheva index 0.187), while recurrent features showed perfect mean effect-direction consistency.

These findings show that strong patient-level discrimination can emerge from an extremely sparse low-montage EEG feature space. The strongest run concentrated its signal in a small recurrent biomarker set centered on temporal spectral entropy and frontal-temporal difference measures, supporting low-montage resting-state EEG as a promising basis for pretreatment stratification of tDCS response in treatment-resistant depression.
\end{abstract}
